## How to Update Credits

This file should be kept in sync with the CreditsData ScriptableObject asset in the project. When adding new assets:

1. Add entry to this CREDITS.md file
2. Add corresponding AssetCredit entry to the CreditsData ScriptableObject
3. Verify license compliance for all assets
4. Test credits display in the game

*This credits file should be updated whenever new assets are added to the project. All asset usage must comply with their respective licenses.*
